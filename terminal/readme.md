# greeting
- /etc/issue : 로그인 전, 콘솔(로컬) 접속시도시 보여줄 메시지파일
- /etc/issue.net : 원격지에서 접속시도시 보여줄 메시지파일
- /etc/motd : 로컬접속, 원격접속 모두에 해당하며 로그인 성공 후 보여줄 메시지파일
